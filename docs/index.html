CYCLOPS' TREASURE

Created for LUDUM DARE 58

Play GitHub Repository

ABOUT

A 2D action-puzzle game where you play as a gnome on a quest to find the legendary Cyclops' Ring. Navigating dark, trap-filled dungeons with limited visibility, you must outsmart monsters and find key items to unlock the way to the next levels.

TEAM

  • Mark Han - 2D Art / Animation / UI / Illustration
  • Mishka PO - 2D Art / Background
  • Igor Vasilev - Music / SFX

CONTROLS

  • WASD - Movement
  • E - Interact
  • 1, 2, 3, 4, 5 - Drop Item

PLAY IN BROWSER

Developed with Unity and deployed using GitHub Pages.
